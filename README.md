#openshare2

这个项目是从100apps大神那里fork过来，然后根据我个人的需求重新修改的版本。非常感谢100apps大神的思路。


因为项目集成了submodule，所以clone代码之后请在终端运行一下代码

```
// 初始化submodule
git submodule init

// 更新submodule
git submodule update

```
然后可以正常使用demo了！
